# XMAZING

Un projet réalisé par Boris Jacquot et Louis Wadbled.

## Installation

### Dépendances

+ SDL2
```bash
apt install libsdl2-dev
apt install libsdl2-gfx-dev
```

### Compilation

```bash
make clean
make
```
